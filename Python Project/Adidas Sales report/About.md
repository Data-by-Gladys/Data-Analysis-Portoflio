# Exploratory Data Analysis of Amazon Sales Performance Using Python
## Introduction

E-commerce companies generate large volumes of transaction data every day. Analyzing this data helps businesses
understand customer behavior, product performance, and profitability. This project performs Exploratory Data Analysis (EDA)
on an Amazon-like sales dataset to uncover important patterns in sales performance, customer purchasing behavior, and product 
trends. Python and data visualization tools are used to analyze the dataset and present insights that can support better business decisions.

## Objectives

The main objectives of this project are to:

a. Perform data cleaning and preparation using Python

b. Explore sales, profit, and quantity trends

c. Analyze performance across product categories, regions, and time

d. Visualize key insights using Python libraries

## Tools Used

Pandas for  data cleaning and manipulation

Matplotlib & Seaborn for data visualization

Jupyter Notebook for analysis environment

## Data Preparation

Before performing analysis, the dataset was cleaned and prepared using the following steps:

a. Loaded the dataset using pandas

b. Checked dataset structure and column types

c. Handled missing values &  duplicate records

d. Ensured numeric fields such as price, quantity, and sales were correctly formatted

## EDA QUESTIONS  

1. Overall Sales Performance: What is the total sales, total profit, and total quantity sold?

2. Sales Trend Over Time: How do sales change over time (monthly or yearly trend)?

3. Category Performance: Which product categories generate the highest sales and profit?

4. Sub-Category Analysis: What are the top 5 sub-categories by sales or profit?

5. Regional Performance: Which region performs best in terms of sales and profit?

6. Profitability Analysis: Are there categories or sub-categories with high sales but low profit?

7. Customer Purchasing Behavior 
• What is the distribution of order quantities? 
• Do customers tend to buy in small or large quantities? 

8. Relationship Analysis 
• What is the relationship between Sales, Quantity, and Profit? 
• Use correlation analysis and visualizations. 

## Key Insights

a. Sales increase when quantity sold increases.

b. Some categories generate high revenue but lower profit due to discounts.

c. Certain regions contribute more sales than others.

d. Most customers purchase small quantities per order.

e. Discounts can increase sales but may reduce profit margins if not controlled.

## Business Recommendations

Based on the analysis, the following recommendations can support better business decisions:

a. Review discount strategies to ensure profitability is maintained.

b. Focus marketing efforts on high-performing categories and regions.

c. Encourage customers to purchase more items through bundles and promotions.

d. Continue using data analysis to monitor sales trends and customer behavior.
