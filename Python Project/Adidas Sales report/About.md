# Exploratory Data Analysis of Amazon Sales Performance Using Python
## Introduction

E-commerce companies generate large volumes of transaction data every day. Analyzing this data helps businesses understand customer behavior, product performance, and profitability.

This project performs Exploratory Data Analysis (EDA) on an Amazon-like sales dataset to uncover important patterns in sales performance, customer purchasing behavior, and product trends. Python and data visualization tools are used to analyze the dataset and present insights that can support better business decisions.
